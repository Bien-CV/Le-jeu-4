Actions de base:

-void Abandonner(): Permet au joueur de quitter la partie.

-void DeplacementValides(t_perso perso, t_coord depValid[][N]): permet de retourner au joueur les positions valides pour 
    sont perso.

-int SelectAction(): selectionne l’action que le joueur souhaite effectuer (exple: SelectPerso / PasserTour…) retourne le n° 
    de l’action à effectuer

Les grandes lignes:
-------------------------

Attaque perso courant : 
************************
Sélection perso -> Sélection skill -> Sélection cible -> Application de la compétence sur la cible -> Animation blinking star 
    lololol -> Vérification des PVs, suppression si mort -> décrémentation du nombre d’actions disponibles pour le perso -> Proposition de l’orientation du perso -> fin du tour automatique

SelectPerso SelectAction 
Mouvement perso joueur:
***************************
Sélection perso -> Proposition des cases de déplacement valides -> Sélection case de déplacement -> Déplacer perso 
    -> Proposition de l’orientation du perso -> Fin du tour automatique.

Tour:
***************************
Sélection de perso ou passer tour ou abandon -> Mouvement perso -> Compétence ( dont attendre ) -> Vérification des conditions 
    de victoires -> Orientation -> fin de tour.
